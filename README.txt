NEON HEIST — Static HTML/CSS demo
=================================

Original concept site inspired by Miami neon / vaporwave aesthetics.
Not affiliated with any existing game or publisher.

Files
-----
  index.html   Landing page
  login.html   Sign-in page (visual only, no real auth)
  play.html    "Loading game…" screen
  css/styles.css
  js/login.js
  js/play.js

How to run
----------
Just open index.html in any modern browser. No build step, no server needed.

Flow
----
  index.html  →  click "Sign in to play"
  login.html  →  enter any username + password  →  Start playing
  play.html   →  fake loading bar
